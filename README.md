# SYSU-online-edu-js

**SYSU 在线教学平台自动播放脚本**

自动播放和跳转下一个视频的油猴脚本。等待页面加载完成后，自动点击播放按钮，检查目标值，然后点击下一个视频按钮。如果检测到"发表论坛帖子"则直接跳转到下一个活动。

<sup>📌 **免责声明**：（感谢伟大的 GPT协助编写程序，Claude整理md）仅供学习交流使用。</sup>

> ⚠️ **注意**：论坛帖子发表任务仍需手动完成

## 📋 功能特性

- ✅ 自动播放视频
- ✅ 自动跳转下一个视频
- ✅ 智能识别论坛帖子活动跳过
- ✅ 无需手动干预

## 🚀 安装方法

### 第一步：安装油猴插件

1. **下载油猴插件**  
   根据教程安装油猴插件（请自行搜索安装方法）

2. **开启开发者模式**（以桌面版 Chrome/Edge 浏览器为例）
   - 点击浏览器界面右上角三横点
   - 选中：**扩展** → **管理扩展**
   - 点击"**开发人员模式**"按钮打开开发者模式

### 第二步：安装脚本

#### 方法一：自动更新（推荐）

1. 打开 [Greasy Fork](https://greasyfork.org/) 网页
2. 搜索"SYSU 在线教学平台助手"
3. 点击并安装

#### 方法二：手动安装

1. 下载本仓库中的 `.js` 文件
2. 打开管理面板
   - 点击浏览器右上角的油猴图标
   - 选择：**管理面板** → **实用工具**
3. 在"**文件导入**"区域，点击"选择文件"，选中上述保存的 `.js` 文件
4. 点击"**导入**"按钮
5. 导入后在脚本列表中确保开关为**开启状态**
6. 刷新匹配页面，脚本会自动运行

## 💡 食用方法

1. 打开心理健康教育中的某个未观看视频
2. 点击播放按钮
3. 脚本自动运行，享受自动播放体验 🎉
